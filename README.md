**DiscordUtils** is a *lightweight*, *discord utility minecraft plugin*, designed to help every server owner *connect* the players with **his discord server**.

---

## ‚öôÔ∏è Features
 - **Fully configurable GUI**, with info items and more!
 - **Custom block** for the *discord-item* in the GUI
 - Fully configurable **sounds**
 - Multiple ways of **displaying the discord link to the player**
 - Toggleable mini **fetching-data** feature
 - **Simple configuration**  
And **more**! Check `config.yml` for every feature!  

üî¥ **DiscordUtils** works best with a *PAPER server*, because it uses some stuff which work **ONLY** on it.  
    For more information, check the release info [here](https://github.com/ItsAndrew1/DiscordUtils/releases/tag/v1.0).

---

## ü™Ñ Commands
| Command                | Description                                       | Permission           |
|------------------------|---------------------------------------------------|----------------------|
| `/dcutils setdclink`   | Sets the discord link                             | `discordutils.staff` |
| `/dcutils setdcchoice` | Sets the way of displaying the link to the player | `discordutils.staff` |
| `/dcutils reload`      | Reloads **config.yml** file.                      | `discordutils.staff` |
| `/dcutils help`        | Opens the *help manual*                           | `discordutils.staff` |
| `/discord`             | Opens the discord GUI                             | `discordutils.use`   |

---

## üìÅ Configuration Files
**DiscordUtils** uses only 1 configuration file, `config.yml`, in which you can _configure_ the whole plugin. Sounds, GUI, tasks and **much more**.
